# Odoo Serial Quotation Workflow

Installable Odoo 18 addon for selecting serial numbers directly on quotation lines and transferring them to delivery move lines.

## Features

- Serial selection on quotation lines
- Product and quantity validation
- Duplicate prevention within an order
- Required-serial validation before confirmation
- Delivery reservation and lot assignment

Module: `portfolio_serial_quotation`  
Dependency: `sale_stock`  
License: LGPL-3

Test in a development database before production use.
